# Pintos Project
## Usage
  You need to Download QEMU.
    https://www.qemu.org/download/
## Project1: User Program (1)
  ### Implementation
  1. Process Terminaition Messages
  2. Argument Passing
  3. System calls

## Project2: User Program (2)
  ### Implementation
  1. System call about file system
   create, remove, open, close, filesize, read, write, seek, tell
  2. File Descriptor
  3. Synchronization

## Project3: Threads
  ### Implementation
  1. Alarm Clocking
  2. Priority Scheduling
  3. Advanced Scheduler (BSD Scheduler)

## Project4: Virtual Memory
  ### Implementation
  1. Page Table Management
  2. Paging to and from swap disk
  3. Stack Growth
